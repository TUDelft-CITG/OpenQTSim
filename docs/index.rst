Open source Queueing Theory Simulation
==========================================

OpenQTSim is a python package for queueing theory simulations.

Welcome to OpenQTSim documentation! Please check the contents below for information on installation, getting started and actual example code. If you want to dive straight into the code you can check out our `GitHub`_ page or the working examples presented in `Jupyter Notebooks`_.
